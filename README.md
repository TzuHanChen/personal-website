## 開發工具

使用 Next.js + TypeScript + Tailwind CSS + Heroicons 製作網頁畫面

使用 Next.js route handler 實作 REST API 讀取 JSON 資料，再串接 REST API 顯示專案資料

使用 Vercel 取得 GitHub 的提交紀錄、自動部署
